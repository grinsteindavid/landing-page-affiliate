## Deploy with GitHub Pages

1. Push to GitHub
3. Go to **Settings → Pages**
4. Set Source to **Deploy from a branch**
5. Set Branch to **main**, folder **/ (root)**
6. Your site will be live at `https://grinsteindavid.github.io/landing-page-affiliate/`
